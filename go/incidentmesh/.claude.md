# IncidentMesh Development Notes

## Testing

Go is not installed locally. Use Docker to run tests:

```bash
docker run --rm -v "$(pwd)":/app -w /app golang:1.21 go test -race -v ./...
```

Or count passing/failing:

```bash
docker run --rm -v "$(pwd)":/app -w /app golang:1.21 sh -c 'go test -race -v ./... 2>&1 | grep -c "--- PASS:"'
docker run --rm -v "$(pwd)":/app -w /app golang:1.21 sh -c 'go test -race -v ./... 2>&1 | grep -c "--- FAIL:"'
```
